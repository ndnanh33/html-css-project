HTML5 Semantic

Sử dụng các thẻ HTML semantic giúp cải thiện SEO và khả năng tiếp cận của website. Các thẻ như header, nav, main, article làm cho cấu trúc trang rõ ràng hơn.

<article> <header> <h2>Tiêu đề bài viết</h2> </header> <section> <p>Nội dung chính...</p> </section> <footer> <p>Tác giả: Nhật Anh</p> </footer> </article>

CSS Flexbox Layout

Flexbox giúp tạo layout linh hoạt và responsive. Hỗ trợ căn chỉnh các phần tử theo chiều ngang hoặc dọc một cách dễ dàng.

.container { display: flex; justify-content: space-between; align-items: center; gap: 20px; } .item { flex: 1; padding: 20px; }

CSS Grid System

CSS Grid cho phép tạo layout 2 chiều phức tạp. Hoàn hảo cho việc chia trang thành các vùng và sắp xếp nội dung theo ý muốn.

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); grid-gap: 25px; } @media (max-width: 768px) { .grid-container { grid-template-columns: 1fr; } }

CSS Custom Properties

Biến CSS giúp quản lý màu sắc và giá trị dễ dàng. Thay đổi một lần ở :root sẽ áp dụng cho toàn bộ website.

:root { --primary-color: #1e3a8a; --spacing: 20px; --radius: 16px; } .button { background: var(--primary-color); padding: var(--spacing); border-radius: var(--radius); }

Glassmorphism Effect

Hiệu ứng kính mờ tạo giao diện hiện đại và sang trọng. Sử dụng backdrop-filter để làm mờ background phía sau.

.glass { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); box-shadow: 0 8px 32px rgba(0, 0, 0, 0.37); }

Responsive Web Design

Thiết kế responsive đảm bảo website hiển thị tốt trên mọi thiết bị. Sử dụng media queries và đơn vị linh hoạt.

@media (max-width: 768px) { .container { grid-template-columns: 1fr; padding: 10px; } .card { width: 100%; } }
